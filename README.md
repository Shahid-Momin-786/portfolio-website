# Shahid Momin | Dual-Role Portfolio

🚀 **Live Website**: [https://portfolio-website-indol-chi-81.vercel.app/](https://portfolio-website-indol-chi-81.vercel.app/)

A premium, dual-role developer portfolio designed to showcase my expertise as both a **Full Stack Web Developer** and a **Flutter Mobile Developer**. The portfolio features a seamless role-switcher that dynamically updates content to target specific job opportunities.

## 🛠 Tech Stack

- **Framework:** React + Vite
- **Styling:** Vanilla CSS (Handcrafted custom design system)
- **Animations:** Framer Motion
- **Contact Form:** EmailJS
- **Deployment:** Vercel

## ✨ Key Features

- **Dual-Role Switcher:** Instantly toggles between Web and Mobile development context.
- **Dynamic Content:** Updates the hero section, subtitles, resumes, and highlighted skills based on the selected role.
- **Serverless Contact Form:** Fully integrated EmailJS for direct messaging without a backend server.
- **Premium UI:** Clean layout, vibrant custom gradients, and smooth micro-animations.
- **Fully Responsive:** Perfectly optimized for all devices and screen sizes.

## 💻 Local Setup

1. Clone the repository
```bash
git clone https://github.com/Shahid-Momin-786/portfolio-website.git
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
Create a `.env` file in the root directory and add your EmailJS configuration:
```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

4. Start the development server
```bash
npm run dev
```
